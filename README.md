# dotfiles

All things setup.

```bash
git clone https://github.com/petecorreia/dotfiles.git
cd dotfiles
./install.sh
```

After it install, open a new term tab and:

```bash
fisher add rafaelrinaldi/pure
fisher add edc/bass
fisher add FabioAntunes/fish-nvm
```
